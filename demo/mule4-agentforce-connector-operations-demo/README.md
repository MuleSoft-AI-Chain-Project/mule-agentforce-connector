MuleSoft Agentforce Connector Demo - All Operations
====================================================
Anypoint Studio demo for MuleSoft Agentforce Connector showcasing all available operations.


Prerequisites
-------------

* Anypoint Studio 7 with Mule ESB 4.6.9 Runtime.
* MuleSoft Agentforce Connector v1.3.0+


How to Run Sample
-----------------

1. Import the project folder demo in Studio.
2. Update the Client Id, Client Secret and Salesforce Org URL in the Connection config
3. Save the configuration & run the application


About the Sample
----------------

This comprehensive demo showcases various Agentforce connector operations:
* Agent conversation lifecycle management
* Message handling and response processing
* Agent variables and context management
* Session management

You can use Postman to trigger requests to http://localhost:8081


Configuration
-------------

Configure the following in your connection settings:
* **Client ID**: OAuth Connected App Client ID
* **Client Secret**: OAuth Connected App Client Secret
* **Salesforce Org URL**: Your Salesforce instance URL (e.g., https://login.salesforce.com)
